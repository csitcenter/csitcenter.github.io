B.Sc Csit Sixth Semester
This semester covers software cycles and testing, compiler design, e-governance, .NET framework, technical writing, hardware design, e-commerce, and automation/robotics.
Software Engineering

$39

This course covers various software engineering concepts, such as software process models, agile development, requirements engineering, design, implementation, testing, evolution, and project management. The goal is to help students develop high-quality software using different management skills.

Compiler Design and Construction

$49

This course introduces fundamental concepts of compiler design, covering lexical and syntax analysis, syntax-directed translation, type checking, and more. Objectives include developing knowledge and skills in compiler design, creating lexical analyzers, parsers, and small compilers using various tools and programming languages.

E-Governance

$39

This course covers various concepts of e-government and e-governance, including models and infrastructure development, security, and data warehousing and mining. Objectives include developing knowledge of e-governance, understanding different models and infrastructure, and implementing security measures and data analysis in e-governance.

NET Centric Computing

$49

This course teaches cross-platform web application development using ASP.NET Core MVC framework and C# programming language. Objectives include understanding both the theoretical foundation and practical aspects of ASP.NET Core and C# language features

Technical Writing

$39

This course helps students improve their workplace writing skills, including listening, researching, planning, composing, revising, and editing documents for practical fields such as business, science, and technology. Objectives include understanding the importance and characteristics of technical writing and producing quality technical documents such as proposals, manuals, briefs, and news releases. The course includes sample documents, critical thinking questions, and writing advice.
